We take notice of the following operations - open, fstat, mmap, read, close
and mprotect.
open(): opening the location of the libraries on the system
fstat(): calculates the size of the library that it needs to link so that it can allocate memory 
mmap(): create the mapping between the library and the memory 
read(): reads the information about the dynamic library.
mprotect(): protect the code 
close(): close the library once code has been dynamically linked
